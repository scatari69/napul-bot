# 🛡️ Deadlock Team Bot

Бот для организации сборов игроков в Deadlock. Помогает собирать пати, вести честную статистику "напуляторов" и организовывать группы в М+.

## 🚀 Функционал

* **Управление командой:** Легкое добавление (`/addme`) и удаление (`/removeme`) себя из основного состава.
* **Умный сбор (`/tag`):** Бот тегает только тех, кто еще не ответил на текущий сбор. Имеет встроенную защиту от спама (Cooldown).
* **Топ-статистика:** Автоматический учет успешных сборов и личный рейтинг игроков, доступный через `/stats`.
* **М+ Сборы (`/mplus`):** Интерактивная система распределения ролей (Танк, Хил, ДД) с кнопками и автоматическим уведомлением о сборе группы.
* **Автоматизация:**
    * Ежедневная очистка списков в 03:00 ночи.
    * Таймаут сбора через 30 минут бездействия.
    * Автоматическая очистка заброшенных М+ сборов.
